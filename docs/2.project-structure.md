# 2号文档：项目结构

## 2.1 目录结构

```text
.
├─ aichan
│  ├─ main.py
│  ├─ registry
│  │  ├─ __init__.py
│  │  └─ server.py
│  ├─ hub
│  │  └─ src/hub
│  │     ├─ __init__.py
│  │     ├─ signal_hub.py
│  │     ├─ signal_processor.py
│  │     └─ registry_signal_trigger.py
│  ├─ agent
│  ├─ core
│  ├─ memory
│  ├─ pyproject.toml
│  └─ uv.lock
├─ gateway
│  └─ channels
│     └─ cli
│        ├─ cli_gateway.py
│        ├─ requirements.txt
│        └─ Dockerfile
├─ docs
└─ docker-compose.yml
```

## 2.2 关键模块说明

- `registry/server.py`
  - 网关注册接口；
  - OpenAPI 工具映射；
  - channel 网关 SSE 感知。
- `hub/registry_signal_trigger.py`
  - 事件总线消费者；
  - 新 user 消息去重触发。
- `hub/signal_processor.py`
  - Registry 通道读写；
  - 动态工具快照构建；
  - Agent 推理回写。
- `gateway/channels/cli/cli_gateway.py`
  - 通道 API + SSE；
  - 启动自动注册重试。
