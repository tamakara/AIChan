## 2.2 关键模块说明

- `mcp_hub/src/mcp_hub/manager.py`
  - MCP 连接生命周期管理；
  - 多服务工具发现、路由与调用。
- `hub/channel_poll_trigger.py`
  - 通道消息轮询；
  - 新 user 消息去重触发。
- `hub/signal_processor.py`
  - 通道读写；
  - MCPHub 动态工具快照拉取；
  - Agent 推理回写。
- `mcp_servers/cli/server.py`
  - MCP 隧道（`/mcp/sse` + `/mcp/messages`）；
  - 通道 API + SSE。
